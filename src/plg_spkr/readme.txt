  Модуль эмуляции спикера

Этот модуль эмулирует динамик, подключеный к одному из портов виртуальной системы.
Ииспользуется только нулевой бит. Если он установлен, напряжение на динамик подано.
Если сброшен, напряжения нет.

Таким образом, устанавливая и сбрасывая этот бит с определенной частотой, можно 
получить произвольный звуковой сигнал.

Аналогичным устройством снабжены многие платы микроконтроллеров. Спикер используется
для подачи звуковых сигналов оператору, сообщающих об ошибке или удачном течении процесса.
